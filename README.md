# Plunketts::Engine

This Rails Engine contains common code user by all Rails applications at Plunkett's Pest Control.

## Usage


## Installation
Add this line to your application's Gemfile:

```ruby
gem 'plunketts-engine'
```

And then execute:
```bash
$ bundle
```
